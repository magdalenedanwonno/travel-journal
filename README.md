## Travel Journal | React & Vite

A responsive travel journal project built during my Scrimba Frontend Career Path. This application demonstrates dynamic data rendering using React components, props, and the `map()` method.

### 🛠️ Tech Stack
* Framework: React
* Build Tool: Vite
* Styling: Mobile-first CSS

### 🚀 Getting Started
1. Install dependencies: `npm install`
2. Run locally: `npm run dev`

### 📂 Features
* Dynamic Mapping: Renders journal entries from a centralized data file.
* Responsive Design: Optimized for mobile and desktop viewing.
* Clean UI: Includes Google Maps links, trip dates, and location photos.
